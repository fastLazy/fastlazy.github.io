镜像源

添加 archlinuxcn 源​

sudo vim /etc/pacman.conf

#文件末尾添加以下两行
[archlinuxcn]
Server = https://repo.archlinuxcn.org/$arch

更新系统​

sudo pacman -Syyu

签名错误解决办法​

sudo pacman -S archlinux-keyring && sudo pacman -Syu

sudo pacman-key --init && sudo pacman-key --populate
